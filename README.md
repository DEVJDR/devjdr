## üîß Getting Started

To get started with your own portfolio website:

1. Clone this repository:

   ```bash
   git clone https://github.com/Arun/minimal-next-portfolio.git my-portfolio
   cd my-portfolio
   ```


2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your web browser to see the website.
